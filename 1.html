HTML이란 무언인가?

HyperText Markup Language 또는 HTML 은 웹 브라우저 에 표시되도록 설계된 문서 의 표준 마크업 언어 입니다 . CSS( Cascading Style Sheets )와 같은 기술 및 JavaScript 와 같은 스크립팅 언어 의 도움을 받을 수 있습니다 . 웹 브라우저 는 웹 서버 또는 로컬 저장소에서 HTML 문서를 수신 하고 문서를 멀티미디어 웹 페이지로 렌더링 합니다. HTML은 의미론적 으로 웹 페이지 의 구조를 설명 하고 원래 문서의 모양에 대한 단서를 포함했습니다. HTML 요소 는 HTML 페이지의 빌딩 블록입니다. HTML 구성을 사용하면 이미지 및 대화형 양식 과 같은 기타 개체 가 렌더링된 페이지에 포함될 수 있습니다. HTML은 표제, 단락, 목록, 링크 , 따옴표 및 기타 항목과 같은 텍스트에 대한 구조적 의미 를 표시하여 구조화된 문서 를 생성하는 수단을 제공합니다. HTML 요소는 꺾쇠 괄호 를 사용하여 작성된 태그 로 구분됩니다 . 및 와 같은 태그 는 콘텐츠를 페이지에 직접 소개합니다. 다음과 같은 다른 태그

CSS를 포함하면 콘텐츠의 모양과 레이아웃이 정의됩니다. W3C ( World Wide Web Consortium )는 전 HTML 관리자이자 현재 CSS 표준 유지 관리자로서 1997년부터 명시적 표현 HTML보다 CSS 사용을 권장했습니다.
