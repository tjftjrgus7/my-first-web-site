WEB

일반적으로 웹 으로 알려진 WWW ( 월드 와이드 웹 ) 는 인터넷 을 통해 문서 및 기타 웹 리소스 에 액세스할 수 있도록 하는 정보 시스템 입니다. [1] 문서 및 다운로드 가능한 미디어는 웹 서버 를 통해 네트워크에서 사용할 수 있으며 웹 브라우저 와 같은 프로그램에서 액세스할 수 있습니다 . World Wide Web의 서버와 리소스는 URL( Uniform Resource Locator ) 이라는 문자열을 통해 식별되고 위치를 찾습니다. 원래의 매우 일반적인 문서 유형은 HTML( Hypertext Markup Language ) 형식 의 웹 페이지 입니다. 이 마크업 언어는 복잡한 사용자 상호 작용을 구현하는 일반 텍스트 , 이미지 , 포함된 비디오 및 오디오 콘텐츠, 스크립트 (짧은 프로그램)를 지원합니다. HTML 언어도 지원다른 웹 리소스에 대한 즉각적인 액세스를 제공하는 하이퍼링크 (포함된 URL). 웹 탐색 ​​또는 웹 서핑은 여러 웹사이트에서 이러한 하이퍼링크를 따라가는 일반적인 관행입니다. 웹 응용 프로그램 은 응용 프로그램 소프트웨어 로 작동하는 웹 페이지입니다 . 웹에 있는 정보는 HTTP( Hypertext Transfer Protocol ) 를 사용하여 인터넷을 통해 전송됩니다. 공통 테마와 일반적으로 공통 도메인 이름 을 가진 여러 웹 리소스가 웹사이트 를 구성합니다 . 단일 웹 서버는 여러 웹 사이트를 제공할 수 있지만 일부 웹 사이트, 특히 가장 인기 있는 웹 사이트는 여러 서버에서 제공할 수 있습니다. 웹사이트 콘텐츠는 수많은 회사, 조직, 정부 기관 및 개인 사용자 에 의해 제공됩니다 . 방대한 양의 교육, 엔터테인먼트, 상업 및 정부 정보로 구성됩니다. World Wide Web은 세계의 지배적인 소프트웨어 플랫폼 이 되었습니다. [2] [3] [4] [5] 전 세계 수십억 명의 사람들이 인터넷과 상호 작용하기 위해 사용하는 기본 도구입니다. [6] 웹은 원래 문서 관리 시스템 으로 생각되었습니다 . [7] 1989년 CERN 에서 Tim Berners-Lee 에 의해 발명 되어 1991년에 일반에 공개되었습니다.
